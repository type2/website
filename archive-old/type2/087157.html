Only time I ever had a problem like that was when I adjusted brakes in 
20 degree weather, then it got to be 80 2 days later....a few stops later, 
I had the same problem.  Backed the adjusters off about 3 clicks and all

This assumes an otherwise healthy set of brakes....no bad rubber hoses, no 
junk in the mc that blocks the "relief" valve.
