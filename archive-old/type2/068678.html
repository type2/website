remove the facia when I leave the bus.  I also modified the original

Here 'xxxx' is a new shelf, and 'y' is the new 'fake' facia.  The CD
changer itslef was mounted onto the shelf, 'xxxx', and as luck would
have it, this leaves the perfect space for storing CD's next to it.

As far as the facia goes, I cut a piece of ply that exact shape, and put
2 long nails in the right hand edge sticking out to the right.;

+--------------------------------------------+
|                                            |
|   0                                        |========
|                                            |
|                                            |
|                                            |
|                                            |========
|                                            |
+--------------------------------------------+

The nails stick out about 2 inches (be careful to drill the ply first,
so it doesn't split!).

OK, now I drilled 2 holes in the side of the new cupboard for the nails
to slot into.

I then mounted one of those pop-in-pop-out magnetic catches onto the top
of the original shelf ABOVE the CD player, and put a washer on the facia
(see '0' above).

Now, to close the facia, push the nails into the holes, then press the
washer closed on the catch - looks like theres nothin there!  Make sure
you hide the wires, as these could give the game away...

To open the cabinet, press 'o' above on the facia, this releases the
catch - TIP here: make sure that the facia itself is as 'snug' to the
wood as possible - makes opening that bit 'firmer' - and means you
really have to KNOW how to open the cabinet to get into it.

Hope this makes sense to everyone.
