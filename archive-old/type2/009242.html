This is doable. Mexican loafs use I4 gas engines which are bolt-
equivalent with the diesels. Conversion would involve getting 
parts (engine mounts, bell housing, flywheel, cooling
system etc.) from a crashed Mexican loaf. Install engine mounts
and engine, run coolant hoses and cut out the nose for a
radiator and grill.
