IF you still have the original distributor in your 76
bus it can be one of five different numbers depending
on the build date of the and the vin number of the
bus. 

If it was built before the end of 12/75 and it has a
dual vacuum unit on it, and it is a manual trans bus,
it should be part # 021-905-205AB or 022-905-205AB

If it was built before the end of 12/75 and it has
dual vacuum unit on it,and it has a auto trans, it
should be part # 021-905-205AC or 022-905-205AC

If it was built after 1/76 and it has a single vacuum
on it, it should be part # 021-905-205P or
022-905-205S
