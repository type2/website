Tom said:
> Also, from what I've heard, the combination LP/AC/DC

> fridges aren't 
> very efficient in DC mode.  So again, if you're
>going for DC only 
>(solar, etc.), look for DC only fridges.

Combination LP and anything else refrigerators are
going to be absorption type refrigerators, and use
whatever electricity they use in electric mode to
repace the heat of the LP burner, so they are usually
very inefficient when run in the electric mode.
So, yes, use a refrigerator that runs only on 12 volts
DC for low power consumption in a bus.
