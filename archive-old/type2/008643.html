This is, I am sorry to say, an excellent excuse for me to
re-iterate the FART "T does not equal Type" mantra :-)

Your bus is a Type 2, but it also is a 2nd generation Transporter,
which Volkswagen chose to call a "T2".  So your request might
be taken either as one for a 1st or 3rd generation Transporter,
T1 or T3 respectively, or as one for a Bug or VW 1500/1600,
Type 1 or Type 3 respectively.

T stands for Transporter.  T does not stand for Type.

My favourite example is that an '81 Vanagon would be a Type 2, T3, 
with a Type 4 engine.  It is not a T2, that's a Loaf.  It is
not a Type 3, that's a VW 1500/1600.  And it does not have a 
T4 engine, which would be out of a Eurovan.

The only way to stop the confusion is for everyone to do it the
right way.  Please.  ;-)
