Megasquirt can be a bit intimidating before you get over the learning
curve, but I certainly wouldn't be *afraid* of it.  I've come away from
my reading up on it that it's a very good system ... and it's certainly
inexpensive!

>  Does anyone sell a complete solution?

Define "complete".  You're going to need some of those D-Jet components
to use Megasquirt... the plenum (which you'll have to modify slightly;
it will need to be fitted with the MAP sensor), the intake runners, and
the injectors themselves (you'll probably want bits of the D-Jet wiring
harness, too).  Other than that, the Megasquirt web site has info about
where assembled units can be purchased, I think.
