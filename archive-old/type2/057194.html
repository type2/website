Interesting idea.  You might have to do some hefty rewiring of the cold 
start valve though to make it work all the time.  You would have to wire 
the thermo-time switch out of the circuit and you would have to somehow 
wire around the fact that the cold start valve only operates while the 
starter is operating.  Then you gotta see if you can get enough gas out 
of it to run a 4-cylinder engine :-)

I think if you tried this the engine would quickly flood out and die.  
Having the injectors held open all the time would result in a terribly 
rich mixture.  In fact, this is one possible failure mode of the ECU, 
holding all the injectors open.  I doubt it would run this way.
