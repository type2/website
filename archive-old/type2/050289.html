> At 06:51 PM 1/8/01 -0500, Chad Yunge wrote:
>> Removing Type IV engine from 78 bus for first time.  It is unclear to me if
>> the transmission must be supported while removing the engine.  What is the
>> best method of doing this while pulling the engine back from the tranny with
>> a floorjack?  Any suggestions on this matter would be greatly appreciated.
> 
> The tranny supports itself - no need for additional support.  Same for any
> 72-79 US-spec bus with type IV engine.

Check first to make sure the two bolts that are supposed to hold the
transmission in the bus are actually there. You'll be able to feel them
right below the gas tank cover plate. The PO of my Riviera took them out,
with the result that when I removed the engine, the transmission came with
it.
