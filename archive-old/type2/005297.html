I'm running a 009 with a 34PICT carburetor.  It didn't even think about
accelerating until I'm set at least 10d BTDC.  I timed it using the
procedure (at full advance) on Boston Bob's web page (I dunno the address)
and it runs very well with no hesitation or flat spot.  They set it for
29-32d at 2-3000RPM.  It stops advancing up there so the exact RPM is not so
important.  I put a mark on my pulley (upright engine) 43mm to the right of
TDC and lined this mark up with the crank case split at 2-3000RPM using a
standard timing light.
