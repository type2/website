Mike:
If you have good compression, there is nothing to be gained by a rebuild.
You say it has a hard spot. Does it also have this hard spot with the spark plugs out? Like when you are checking the compression?
I think your timing is set wrong, like 20-30 degrees too advanced. when it does fire, it is trying to back the engine up!
If  you could get a Muir book, try to set the timing by the static method to get it started. Then use a timing light to set it while it's running.
