Sounds a lot like the master cylinder is at fault.  The pump-up in 3-4
strokes is the main clue I'd base that on.  The difference between the
front and back should be because one part of the master cylinder is
worse than the other.  What were the symptoms that prompted you to do
the complete brake job in the first place?  Sounds like you've exhausted
the resources available-- thanks for doing the obvious stuff first!  I
had a similar problem with a Rabbit, and it took a new m/c to get it
back to working right.  And a custom mod on the one I bought because the
P.O. had changed out the vacuum booster, which had a longer pushrod on
it, not letting the m/c return fully.  Might want to check that, the
pushrod should not contact the M/c until you actually push the pedal
down.  If it rides on the m/c piston, it will cause problems not unlike
the ones you have, but will lock up the brakes if you aren't careful.

HTH.  :<)
