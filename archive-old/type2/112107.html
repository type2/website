---When you remove the ground cable and then go to put
it back on... are you seeing a small spark jump? If so
then there is to much current being drawn. Is
everything turned off? If you have a aftermarket
radio/cd player it will draw current to keep the
memory active.. but very little..
  Spark or no spark you need to do what Karl wrote
(below).
If you arent familure with a VOM then you might want
to
look at this..
-http://www.geocities_dot_com/magicbus1974/volts.htm
If you see current being drawn then start taking fuses
out one at a time and see if removing one of them
makes
the current flow stop. That will tell you which ckt 
has either a problem or a device like your CD player
which requires constant voltage for memory. But you
are looking for a higher current draw than what a CD
player alone draws.
