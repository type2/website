getting its accessories bolted back on, and I have to make a decision about
EGR.  I have all the parts, and have cleaned out all the appropriate tubes

blocked, it would run rich when I unblocked it for the smog check.

B) If I get the mixture adjusted for the smog test, it would then run too
lean after the smog was done and I block off EGR again for daily use.

Does anybody have any idea if:

Plan A) would be too rich to pass smog?

Plan B) would be lean enough to harm the engine?
