> The bus now starts fine but stalls when ever not feeding fuel. Acceleration
> feels very heavy-  Idles fine when first started and then after warm idles

This part sounds kinda like a fuel delivery problem, but a few months
ago I got a rebuild AFM for my bus. Turns out that the machine they
apparently calibrate it with was bumpkus. I had the same symptoms when
changing it out. I didn't run it long enough to tell if it was running
rich from the plugs, but the horrible and slugish effects were about
the same.

All the same, you really should test the FI system per the bentley. It
really only takes a FI pressure gauge in addition to your regular
tune-up tools. Try to verify if the system is getting enough pressure
and if not why not (clogged tank/filter/bad regulator). If you
eliminate this you really help eliminate a lot of the FI system as
being suspect. Bentley has a fantastic section on this.
