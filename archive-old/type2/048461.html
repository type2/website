Well, the system was already established, parts were out there in the
bins, and the pipe sizes were arbitrary anyway, so why change horses in
midstream?  Nobody actually specs a precise fitting size unless they are
designing rocketships, so they just select the next larger standard size
and use it.

If we want to feel fully metricised, we can just assign metric dimensions
to the English-sized pipe, by multiplying the inch size by 25mm.  Hence,
a two-inch fitting becomes a "metric" 50mm fitting with no actual change
in the hardware.
