If you pushed the clutch to the floor, you WERE able to start the bus,
yes?

If so, the clutch is NOT the culprit here.  Transmission related.  If you
tried to start the bus, and all it did was try to move (ie: clutch did not
disengage when pushed...) then I would look at either a frozen clutch
plate, or cable problems, etc..

another check would be to just push the clutch in. No brake. Nose is
downhill, right?  The bus should start rolling.  No? But you can start the
engine? Tranny.  Cant start the engine? clutch. (or both..)
