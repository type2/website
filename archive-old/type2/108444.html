Mine was a little tight last time. I used some antisieze when I installed
the new one. Try using a torque wrench to remove it. If it takes a lot of
torque without moving, stop! (I never found a spec, but assumed similar to
spark plugs)  Then maybe run the engine to get it hot. Things usually
(depending on material) come apart more readily hot than cold. DON'T
re-install while hot; let cool all the way down first(couple hours).
