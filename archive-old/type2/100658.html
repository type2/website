certainly the OEM supplier.  It is highly unlikely that Bosch actually makes
these seals themselves, as they're an electrical manufacturer not a rubber

seals.  I can't see spending $40 a set (!!!!) for the Bosch name in this
case. Continental is a quality outfit and I've never had a single complaint
about their seals.

My price for the Continental set (part # 311 198 261A) is $4.25 for a full
set of 8 (four large, four small).  This is cheaper than carmadillo's
no-name ones (which may be Continental as well).
