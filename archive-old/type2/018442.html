Melissa:
The 1600 that came new in your 71 was, for all purposes, a type 1
engine. It was equiped with waht is called a "universal case" which
provides the bosses and the holes for the 3 bolts which attach to the
rear engine hanger (called a mustache bar cause of its shape). The front
piece of engine tin and the heat exchangers and muffler are different
>from bugs, too. Some bugs came with the bosses on their cases, many bugs
now contain engines rebuilt with universal cases, so it's possible to 
install an engine from a type 1. In addition, there's an adapter bracket
available as well, but I'm not a great fan of them, but it might make a
but run where it wouldn't otherwise.
