The fuel flow direction is pump --> #3 inj. --> #4 inj. --> cold start 
valve --> #2 inj. --> #1 inj. --> pressure regulator. If you clamp the 
input side of the cold start valve you would run full pump pressure to 
cylinders 3 + 4 only. 
Perhaps the Bosch manual is refering to an injection system that doen't 
have a cold start valve in series?
