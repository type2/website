I heard about something like that on type I engines, but I have never seen
reference to it on a type IV.  This includes ALL of the manuals and written
documentation I have ever come across, and my own experience with the type
IV.  Therefore I am reasonably confident that there is no retard built into
any cylinder on the type IV.  I can hardly believe that VW would even do
such a thing - it is a ridiculous way to "solve" an overheating problem.
