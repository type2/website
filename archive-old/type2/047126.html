71 and prior Buses used a lower compression version of the Type 1 engine.  72
and later air cooled buses used an engine based upon the Type 4 sedan series.
Technically, there has never been a "Type 2" engine until the waterboxer.

Cooling will be a big problem.  Type 3's have a rubber boot sealing the fan
inlet to the cooling air inlet ducts.  Look inside of a type 3 engine bay some
time.  Get the owners permission first, I'd say.  Besides, a 1500 in a 74 bus
is sort of going backwards, isn't it.  The westy is pleanty heavy empty, you
know.

> does anyone have experience with these

I restored a Squareback several years ago.  You could convert the Type 3
engine to an upright (type 1) configuration.  At a minimum, you would need to
change the oil cooler to the upright style, Generator stand, block off the oil
fill hole and add a dipstick tube, and you need all of the other type 1
pieces.  New universal cases can be built to either type 1 or type 3.
