I'm not sure if Compu-Fire makes a unit for a vac distributor. I'm =
running a Pertronix Ignitor on my '79 bus with stock vac dizzy. The =
grommet that fills the square condensor hole in the side of the dizzy is =
round, but it goes in okay with a little lube and elbow grease.

I installed the Pertronix in about 1996 or so. Runs great, no problems. =
I think that you have to order the Pertronix or Compu-Fire for a stock =
vac dizzy, because the dealer will usually assume that you're running a =
009.

There are other vac dizzies out there that should work if yours won't. =
Consult the Type2 archives.

P.S. Once it's running, be sure to time it with a strobe light at RPM. A =
Type I engine should advance to about 30 degrees BTDC _at_ 3,500 RPM.
