I'm not recommending attaching the can to the engine!  The point is
that using the soda can as a test object minimizes the number of
variables that can muddle the test data.  You have a controlled heat
source (current through a resistor), a consistant size, shape, and
mass, and only one variable to change.  You could do this with an
engine, but it would be a lot harder to get the test conditions all the
same.
