Hi Tony,

If you want to know the original colour of your Westy, you can find it
out by reading the M-code plate which is fixed on the metal wall behind
your drivers seat. The first four numbers on the third row indicate the
paint color. This is what a regular M-plate looks like (for a '70-'79
baywindow):

## ### ###
### ###
XXXX## ### ### ### ###
## # #### ## #### ##

The 'X' mark the spot where the color code is. I made a list on my
homepage of these codes, if you look it up you will hopefully find out
what color was used.

http://www.tech.nhl.nl/~molen502/bus/t2colors.html

If your code is not in the list, let me know so I can ad it.
