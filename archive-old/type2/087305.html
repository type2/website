Yep, that's it.  The letters "AB" at the beginning of your code means that it is a replacement case built since 1973 in either Mexico, Brazil, or Germany (according to my manual).  It is a 1600cc engine. This means that you have had a new engine installed, or at least they installed a new case. So technically, you have NEITHER an original Bug nor Bus engine, but a re-built engine from the case up, although they may have used some other bug (or bus) top end parts such as heads, etc..  This is the same code as my 71 Westy.  So the good news is that at some point in time you have had a complete re-build.
Hope this helps.
