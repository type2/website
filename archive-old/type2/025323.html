Jack Dinan sends:

> I'm about to install a Petronix and a CD ignition system on my 1776.  The
> instructions say that spark plugs can be opened up quite a bit from the
> stock value.
> 
> What is the theory behind this?
> Is it necessary to do so to run with the Petronix and CD? Or can I leave
> the gaps as they are?

The CDI puts out more power than the stock ignition, so it's capable of
making the spark jump a wider point gap than the standard ignition.

A wider point gap gives a hotter spark, which means more efficient burning
of the fuel, which means "free" horsepower since all the fuel you're putting
into the cylinder can be burned. It's not necessary--you can run a .028"
stock gap with this system if you want--but I'd do it anyway. Like Bluto
said in "Animal House," don't cost nuthin'.

I run a MSD at .050" gap. They tell me you can go wider, but this works for
me.
