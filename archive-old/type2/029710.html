this is similar to a "cheapie" turning brake setup on a dune buggy.  The
idea is that if you have a wheel spinning you can use an individual wheel
brake to stop that wheel, allowing the other wheel to get some torque.

the more "elegant" version of this has individual hydraulic master
cylinders for each wheel.  the "regular" master cylinder feeds into this
so that the brakes work normally, but when the turning brake master
cylinder is actuated it only stops one wheel.

this approach takes more plumbing, but compared with the extensive
metalwork involved in the emergency brake  conversion the hydraulic option
actually looks easier to implement!  It also will work better because it
actuates both brake shoes equally, which the e-brake does not do.

This mod might be useful for a real off-road fanatic, but the effort would
be way more than the increased utility for 99% of bus drivers!
