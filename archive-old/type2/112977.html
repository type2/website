Open call to bussers for a rafting trip in northern CA!

Where:  http://www.malode_dot_com/
When:   July 7-10  Mon-Thurs

Arrive in camp on Mon 7th.
Chili Bar run morning of Tues 8th.
Gorge run afternoon of Wed 9th.
Leave morning Thurs 10th.

Folks can come for all or part.  Each rafting trip is $89 (looks like

gorge run a bit hairier, but nothing that a quasi stalwart first timer
can't handle.  For more info, see the website.
