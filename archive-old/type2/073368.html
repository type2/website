I have put about 100k on my bus with five different sets of heads and never
had a valve seat drop.

Set #1 was what were on the bus when I got it.  History unknown.  Exhaust
valve broke off but the valve seats did not budge.

Set #2 was a set of 1700 heads rebuilt by a local machine shop with lots of
A/C VW experience.  Those heads were doomed by a spark plug thread insert
that came out.

Set #3 was from the junkyard, nice-appearing set of 1700 heads, history
unknown.  I put in new exhaust valves and ran them until I sold that engine.

Set #4 was a pair of GEX rebuilt 1800 heads.  Put about 55,000 miles on
them despite problems from the very beginning with munged spark plug
threads and stripped air injection port threads.

Set #5 is currently on the bus - they are 1700 heads rebuilt by Mark
Stephens with a stage one job done on the combustion chambers (unshrouding,
polishing).  No problems, 11,000 miles and counting.  I expect to have
these heads for a long, long time.
