Yes, as the aforementioned diagram shows, you can add a catalytic converter
to the 49-state setup, simply by putting it in place of the straight pipe.

However, while this would add a cat, it would not in fact make the setup
identical to that used in California.  There was a different layout in
California for '75-78, which used a smaller drivers-side heat exchanger with
the cat converter tucked along side of it (facing front to rear). The
crossover pipe was also different to accommodate this setup. The actual
California setup would require using that unique left side heat exchanger
(which we do carry but it isn't cheap) and the California-only crossover
pipe (which is no longer available).   I can't see any reason to switch to
that particular setup on a non-California bus, though.
