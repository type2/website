I'll go a step farther and say that motors with a mass-air sensor will adjust to displacement as well.  The only problem you will run into is that with a larger displacement, you could run past the operating range of the mass-air meter and/or computer at very high RPM.  The L-Jetronic is mass-air (L for luft (air), right?) and the D-Jetronic is MAP (d=drueck (pressure), uses manifold vacuum to meter fuel).  Any bus with L-Jetronic should adjust very well to displacement changes, while those with D-Jetronic would run lean with a larger than stock displacement.  I can never remember which years came with which system.
