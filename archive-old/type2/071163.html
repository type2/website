Yes, it sure can!  If the snap ring on the input shaft that holds 
the coupler (with its external teeth for the reverse idler gear shaft) 
pops loose, then the coupler can push up the input shaft.  If it
does so then the teeth no longer contact the matching gear on
the reverse idler gear shaft and, Viola!, no more reverse!
