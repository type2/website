I suspect that it would get too hot for silver solder, I have done a
pretty fair amount and I don't think it would hold long enough to get
through the emission inspection

personally I think that if you have to pass on welding that I would make a
"Plumbing" type of connection using mechanical components to lock things
together, it might not be pretty but it would work.  you can pick up a
"pipe flanging" tool at home depot for less than $20 and literally make a
copper pipe connection.  

later after passing the inspection if you wanted to save the pieces for
the next inspection you could make a bypass or "dummy" pipe to fit the
location and just tighten it into place
