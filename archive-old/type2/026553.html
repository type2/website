If these guys have a good reputation with the locals for quality, I'll
believe you.  But I can tell you they DON'T have a reputation as a
reasonably priced shop.

Unless you put a rod through the side of the case, it is unlikely you need a
"new" engine.  And if you do, I think (others may have more accurate numbers
here) that a complete type IV long block engine can be had with warranty for
around $1200 - $1500.  That means that the shop is charging you $900 - $1200
to remove the old engine and install the new one and make sure everything is
running right.  For a competent shop, that process should be about 4-6
hours.  So I think their estimate is way too high.

If you order a new engine (not from GEX!!!!!!), you CAN install it yourself.
Engine removal/installation is not a big deal in a bus as long as you have a
place to work on it, some basic tools (including a floor jack for rolling
the engine out from underneath), and the desire to get greasy.
