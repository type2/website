todd, My understanding of this is different than yours. I have always
understood it as this. Then first assembling a vw engine and first run
in you must run it for 15ish minutes to allow for the lifters and the
cam to work harden against each other to the same hardness. this
should create a situation that both parts have the same hardness and
should wear little against each other and at the same rate. A new
lifter will be quite soft compared to the old camshaft that is already
hardened in the engine, and that will most likely wear. of couse YMMV
