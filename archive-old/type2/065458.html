Okay this might not be advice but another question.  I'm not a 
particularly adept mechanic.  But we had on a Chrysler a compression test 
that showed one cylinder with a really bad compression while all the 
others where fine.  Started make preparations for a new engine and 
continued to run the car.  It was fine.  The compression problem healed 
itself.  Another mechanic checking the car explained that what probably 
happened was a fluke lining up of the rings so the splits were synched.  
This prevented the cylinder from having good compression but with more 
use the rings slipped back out of alignment and compression was fine.

Can the same thing happen with this kind of engine?
