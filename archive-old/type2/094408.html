I found you have to be careful not to file the nylon rider. I used a metal
double sided fingernail file (Jonel fine/medium) from the cosmetics section
of the grocery store.  Because it was so thin and flat I was able to use a
sawing action.  A really big pit would take a long time to file but with the
small pit I had, the file evenly did both sides pinched by the spring action
of the points. I use Bosch points if there is any doubt.
