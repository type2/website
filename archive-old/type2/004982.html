Subject: Re: [T2] Tranny innerchange

Yes it can be used, in almost every type of interchange, as long as the
bellhouse matches the engine, and clutch system you are using
example:
a 79 trans will fit into a 73 bus, if you swap the bellhouse, from the 73
trans, and use the 73 clutch components, and starter.....

One problem I have had is installing a late bellhouse and clutch set up, on
a 74 bus, with the 002 trans....it will work, but is not fun, as the rear
hangar mounts have big clearance issues....I did it in my bus for the 228 mm
clutch, for the Powerstroke engine, and pulling big loads..
