Jeff,

It is barely  possible that your coil/condensor are so marginal that the
power take-off from the meter make the difference between starting and
not starting, but it is very unlikely.  If the bus will start w/o the
meter hooked up, start it and attach the leads while it is running.  If
that kills it, then fersure you are hooking the leads up wrong somehow.
