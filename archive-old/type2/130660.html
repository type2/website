>Regarding idle mixture... Normally this is one using an exhaust gas

You don't know how it's going to change...

>Should I make sure the idle mixture is set correctly *before* I install 
>the CDI, and revert to stock ignition whenever I adjust the idle 
>mixture?

Setting it as as starting point is a good idea but you don't need to revert in order to reset the mixture later.

>Is this going to result in running too lean?

Maybe.

>Or, should I 
>set idle mixture to the %CO spec with the CDI installed?  Or is this 
>going to result in running too rich?

You have to do this afterward the CDI install, first for smog tests, second to dial in the ignition system to get the best mix of economy and power at WOT. You should also have a CHT gauge to give you ball park figures of the changes to head temps.
