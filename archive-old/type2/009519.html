Chuck:

Before trying to start the Bus, you should do the following:

1) Change the oil.
2) Remove the spark plugs, put a teaspoon of motor oil in each plug hole.
3) Using a wrench on the pulley nut, spin the engine a few times to make
sure everything moves, then install the spark plugs.
4) Adjust the valves.
5) Replace all fuel lines (you may need to have the tank cleaned)
6) A complete tune-up might be a good idea.

As long as nothing was wrong when the Bus was parked, it should start up
now with a new battery.
