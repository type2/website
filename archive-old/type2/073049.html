>Im getting ready to install the engine back into my 71 single cab. What I
>want to know is, should I install the engine hanger/support before or after
>I hook the engine to the tranny.

Before. You should install it before you install the muffler too.

>I've always done it before, but i've always
>had trouble getting the hanger to mount up. A crowbar always helps, but its
>still a pain.

Should be a breeze. Attach the rubber engine mount zoobees to the ends of
the moustache bar before you go to install the engine. Tighten the zoobees
down last, after the engine is bolted to the trans. In other words, the
trans is elevated slightly when installing the engine, then the whole
driveline is "settled" into the engine mounts. This is another good reason
to use a bottle jack under the trans, NOT a chain like Muir says to do.
