It might be that spinning the fan faster won't flow more air.  If the flow
is already "choked" with the stock fan pulley spinning it faster won't
make any difference.  If you have a way to measure air flow it would be
easy to determine by plotting air flow versus engine RPM.  If the curve
still has positive slope at 5000 engine RPM then spinning the fan faster
would be effective, but if the curve flattens out it would just cause
problems.
