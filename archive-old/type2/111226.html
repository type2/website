Just for kicks. Double check the spark wires. to make sure they are
still attached in the correct order. #4 should be next to the correct
#1 wire & #2 should be on the other side of the #1 wire when
confirmed.

> I rechecked vavle clearance, (fine)
> someone told me maybe my O ring washer on the intake
> is smashed and maybe blocking it somewhat, is this
> possible?

I doubt it would block the airflow enough to do any restriction of
merrit, however if it is not sealed correctly you will have a huge
vacuum leak which will cause the engine to run HOT. If worried about
this do check it.
 
> I just want my bus to run like a VW should and Im

don't get too frustrated. if some more simple things like making sure
wires etc are in order doen't work, I'd recommend an engine pull. that
way you can confirm a lot of stuff.
