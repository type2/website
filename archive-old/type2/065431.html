I called VDO today, and the odometer is indeed dual purpose (regular
odometer plus trip odometer, switchable).  I can't blame the guy at Summit
for not knowing because it doesn't state that anywhere on the packaging,
according to the VDO guy.

This is an electronic speedometer, so it requires some sort of sensor to
drive the speedometer with electronic pulses, similar to how the tach works
I guess.  My bus has a regular speedometer cable, but I know that some
later model breadloaves had a sensor on the cable for the EGR light, and
I'm pretty sure I could adapt one of these to drive the VDO speedometer.
As soon as I find one in the junkyard, I'm gonna purchase the speedometer
and install it in my bus.

The guy at VDO said you set the speedo by pressing a button, driving
exactly one mile at any speed (can do this at those freeway speedometer
checks) and pressing the button again.  The gauge will take it from there.
Slick!  So is anyone else interested in this?
