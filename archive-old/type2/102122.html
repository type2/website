GD Engine:

> Dwell 44-50 degrees (new points), 42-58 (worn points)
> Points gap 0.40-0.50mm (0.016-0.020)

All that info is nice but list advice says to set the gap so the dwell reads
50 degrees.

Spark plugs: use Bosch W8CC and set gap to 0.7mm

Valves: 0.006 in. on both intake and exhaust on stock 76 2L with solid
lifters.
