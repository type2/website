www.type3_dot_org is a great start.
There's also a forum + loads of other stuff.
The long block of a type 3 is almost identical to the beetle motor (for
1600DP's that is), the main difference is the filler is a tube coming from the
sump for a type 3 + some extra threaded holes for the rear engine support.
The tinware, fan, generator, dissy, carbs etc are all different and not
interchangeable. If you've owned and loved kombi's and beetle's, type 3's are
a great car to set up to as I've found out now I have a couple :)
