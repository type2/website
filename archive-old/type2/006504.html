Steve,
The type 1 case is far from the same alloy as a type 4! The type 1 case is
msostly magnesium, the type 4 case is pure aluminum alloy. When milling a
type 1 case, that boring bar cuts through the type 1 case like hot butter,
if you mill on a type 4, be prepared to change /sharpen the cutter head
after each spigot opening,

A good built type 4 will have that counterweight crank, and maybe even type
1 rod journals, and the same rods that you have in the type 1.

A type 4 engine is much better suited pulling the load of a heavy bus.No
matter how modified a type 1 just cannot make the grade for a period of
abuse that a bus puts the engine through. This is a general statement and
before some one says ":My type 1 has 200 K on it". I would consider you a
veruy lucky person, and ask you to pick my next lottery numbers.

There are many myths about a type 4, and even a type 1, I build both engines
and can tell you that rarely do I ever see a type 1 powered bus, go past 40
K miles, and a type 4 can add 100,000 on that number....

My powerstroke engine allows take off from a stand still in 2nd gear, fully
loaded with 1500- almost 2,000 pounds of parts when we go to car shows. I
have successfully ascended the longest grade in the smokey mountains in 4th
gear, and passed an entire caravan of type 1 powered busses while doing it,
some of which were in 2nd gear, turning 5K . One of these busses was
equipped with a full BERG 2007 engine, he could not use his power, because,
"it would get hot"

Please do some more homework on the two engines, before you jump to
conclusions and compare them.You may want to start your research in a local
junkyard, see how many type 1 engines you see with rods through the case,
and compare that to the type 4's with the same problem...

All us aircooled guys are cut from the same mold, we should all be friends,
no matter what type engine we have. Some are type 1 guys, some are type 4,
and some are both, like me...BUt I will not have a bus with any type 1
engine installed in it..
Jake Raby
