I had this same problem with the replacement seal on my luggage rack too.  I
found that if I pulled the seal down all the way around so the gap is
smaller, the noise and vibration went away.  I pulled it down so it is just
touching the roof but not so far that it falls off the fiberglass luggage
rack.  This did the trick and it has not come loose in the several months
since I adjusted it.
