The Happy99 program is a virus, a worm to be more precise. The virus sends
itself as an attachment in a second email message whenever the infected
machine sends an email.

While this worm will not harm data, it will bring a LAN to it's knees.

DO NOT OPEN the happy99 file.

Here is how to eliminate it if you were infected:

1. Search your C: drive for files "ska.exe" & "ska.dll".
2. Delete them.
3. Go to DOS and delete "wsock32.dll" file
4. Rename your "wsock32.ska" file to "wsock32.dll".
********** 
 If you don't run the
program, you have nothing to worry about. Mac users are unaffected.

As has been said here around these parts recently, all you need to do
is run this program once; from that point on, it sends itself to
every address you send email.
