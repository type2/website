If you moved the plates a full spline then you probably have the rear
end adjusted too high.  This can run the CV joints at too much of an
angle which will cause the problem you described.

You would be well advised to redo the spring plate adjustment, setting
the plates to the correct angle by moving both the inside and outside
splines.  The Bentley book has a nice procedure for this.

BTW, on a loaf bus, you _shouldn't_ be able to see the horizon in the
inside rear view mirror.  If you can see it, the rear end is too high!
