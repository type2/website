I had dieseling with the single weber and it turned out to be the 
valves needed adjustment. The weird bit was the engine would run backwards!

The best way to check the mixture is through analysing the exhaust 
gas. Failing that you have to use fairly crude techniques to know 
you're close to the mark. If you're way too lean you will get 
backfiring through the carb(s). Closer to the mark you can check by 
adjusting the "lean, best idle" (at least for the idle jets) - see 
below. In this process you work out the number of turns the idle 
mixture screw needs to be screwed out to achieve smooth idling. Too 
far out and you are running too lean (and vice versa).

http://www.redlineweber_dot_com/html/Tech/carburetor_set_up_and_lean_best_.htm
