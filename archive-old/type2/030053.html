If you install the jumper between two pins as I suggested in an earlier
post, the relay two relays are electrically identical.  The 4-pin relay
has the jumper connection built in, so the relay is only switchable when
headlight voltage is present.  The earlier relay could be switched even
with the headlights off, which  is why it needed a second voltage source.
