VW engines run like crap at 14.7:1. Stoic is chemically correct, but not
right for our engines.

if you swapped the carbs and had to run leaner jets I will agree that in
YOUR SITUATION (which is rare) you experienced what you experienced. But, if
you used a different set of carbs, jet size is irrelevant, since you used
different carbs (even if they were the same size and model). Tuning hundreds
of cars has shown to me that even the "same" combination uses different jets
on different engines.

I have even had some engines that needed different jets on each CYLINDER.
