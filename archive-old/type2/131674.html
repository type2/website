>Muffler Bolts - bolts, nuts, washers?  Easy to fake a replacement, but I'd 
>like to get it right.

Muffler to heat exchanger - self-locking nut M8 - I would say the pitch is 
1.25 and the length about 20mm. If this is right, which you could mock it up 
to make sure before you buy all of them the bolts, would be M8 X 20mm X 
1.25, Nut would be selflocking if you can find it M8 with the same pitch 
1.25 and an M8 sized lock washer.
