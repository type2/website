Steve:
I don't remember about the spring, but unless you are putting in a new fork,
be sure you check for uneven wear where the fork contacts the TO bearing.
This is a common cause of clutch chattering  when one side is worn more than
the other and RIGHT NOW is when to fix it. I've just smoothed them off with a
hand grinder, but a file would suffice if you can put the fork in a vise.
