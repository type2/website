	How I do it is using an open end - box end wrench.
	I stand the wrench on end so it is in line with the axis of the
bolt, and the open end 
	prongs are straddling the head of the bolt. Then I stick a
screwdriver through the 
	box end ring ans now have a sort of T handle wrench. Cheesy ASCII
below

	                  ||
	                 |   | Box end ring
	TTTTTTTT---|---|----------   Screw driver
	                 ||
	                 ||
	                 ||   Wrench
	                 ||
	                 ||
	               |     | Open end "fork"
	               |[***]|
	                 |  | Bolt   ->  [***] = bolt head
	                 |  |
	                 |  |
