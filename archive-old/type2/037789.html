Jen, regarding battery set ups, one excellent source of parts and information is West Marine, online at www.westmarine_dot_com ( or call 1-800-BOATING for a catalog).  The system I imagine you'd want to set up is very similar if not identical to what you'd find in  boats (my other love).  The West Marine catalog has great mini-articles on various topics, including 12 volt electrical system issues.  At a minimum, you'll want a deep cycle marine battery  ( I'm assuming you want to run house systems and not just set up a reserve starting battery) and some type of isolator to
separate the batteries.
