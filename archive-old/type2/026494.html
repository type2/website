I don't know anything about Dells (I run dual Webers), but what you describe
sounds exactly like intake leaks.  Did you use gaskets and/or sealant
between the manifolds and the heads?  

To check for air leaks, start the engine and get it idling then carefully
spray some carb cleaner around each manifold base.  If the idle changes,
there is an air leak there.  Spray lightly because you want the stuff to be
sucked into the leaks more as a vapor.  If you saturate the area, wipe it
off and try again.  You can also check the area where the carbs bolt to the
manifolds with the same method.

Hope this helps.
