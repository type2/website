We're also careful to not use the same oven for cooking after using it for
casting.  Residual vapors in the oven can taint food cooked in the oven.  I
don't know if powder coating is a similar substance to polyurethane resin,
but there might be a similar reaction.

It takes time to feel the effects.  You'll be able to get away with quite a
bit of double use before you start to exhibit symptoms.
