Hi John,

The problem is that using the coil wire as the trigger will connect  
the batteries as soon as the ignition is switched on.

If the aux battery is dead then you will be drawing extra power from  
the main battery at the same time as you are trying to start the bus.

If you use the blue wire then the relay won't connect the batteries  
until the alternator light goes out, which doesn't happen until the  
engine is running.

It'll work from the coil, but it'll be better for the main battery  
and starter if you run from the blue alternator wire.
