Subject: RE: I need equivalent color for 71 Westy [Chianti Red]

There is a code associated associated with the color "Chianti Red", code L82
or something like that.  Behind the driver's seat is/was a silver colored
sticker with black lettering that will have that code.  Or, you should be
able to find it on the WW site.  In any case, any auto paint supplier should
be able to match colors based on that code.  That's what I did when I had
mine painted.  True, there will be an aging affect that will result in a
difference between the old original paint inside and your new paint outside,
but I found this to be very slight after I repainted my Chianti Red 71
Westy.
