If you've got a '78 then you should have a type 4. Up to 71 buses used an
'upright' bug engine, 72 on used a 'suitcase' type 4 with the occasional
upright being used in mostly european buses. Look under the hood, see a big
fan housing, ya got a upright, dont see a fan housing ya got a type 4. I
doubt bugpack offers one for the type 4 but could be wrong. If they do let
me know as I'll need a type 4 exhaust system soon too.
