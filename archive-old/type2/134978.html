185R14 is a European size designation that actually translates to 185/82R14
(The 82 is "assumed." Why? I have no idea.)  The closest size that uses a
middle digit would be 185/80R14, which is not a common size (although
Vredestein make it).  That size is within 1/8" of a 185R14 - close enough
that my Vredestein distributor actually inventories it as a 185R14.
