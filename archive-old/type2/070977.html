Hi Troy (et al),

First thing is that if a fuel leak only occurs when the engine is running 
then it must be in a pressurised part of the fuel system ie after the pump.

It may be that the only way to find it is to run the engine - but do that as 
a last resort.

With FI there will be a fuel return loop - this allows surpluss pressure from 
the fuel distributor to be returned to the tank - it prevents the fuel in the 
fuel distribution system getting hot, vapourising and causing air locks. If 
the leak appears to be "from the tank" but only when the engine is running 
then I would look at that part of the fuel circuit first.
