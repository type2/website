Hi,
If in doubt, then jack up the rear wheel and with the fronts bloicked, take
off the E-brake.  Then see if there is any movement if you try to tilt/wiggle
the tire with one hand on each side, if more than 1/16" of an inch then the
bearings are probably bad.  To make sure, drain the reduction gear box and see
if there are any chunks of steel in the oil.  Anything bigger than 1/32" dia.
and the more the worse it is.    Eric & Barb
