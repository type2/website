Subject: Re: [VB] 67 Flywheel confusion HELP ME!!

It sounds like your 67 bus had an earlier tranny in it.   As it is usual for 
the tranny to have been replaced in an old bus, don't freak out.    It is 
likely that for some reason a PO used a 6V FW .  annoying.   This means it 
probably has a 6V starter, maybe or maybe not, with a 12V solenoid.   So pull the 
starter, and check to see if it is 12V or 6V.   You probably need to buy a 12V 
starter, a 12V into 6V tranny starter bushing, install them, and then it should 
work.    * Both *  cylinders of the starter should say 12V.   

The 12V FW teeth don't mesh with the 6V starter teeth, you see.   

Did you compare the old FW to the new FW?

Maybe you can get the starter out of the bus the motor came out of.
