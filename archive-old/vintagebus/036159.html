Hi John,
We have only run the early 40 HP engines with the .008" & .012" valve
adjustments.  The very early 1500 had to use the same due it using the same
earlier long studs that retain the rockers assemblies.  So unless you have the
square bossed later heads or the short repair studs in the earlier round
bossed long stud heads.  You would be better off with the .008" & .012" valve
adjustments.  Eric & Barb
