Ronnie wrote:

I have a type 3 cooler on my bus, does cool better then a stock type 1

doghouse, I run a VDO temp switch in the boss(even though i don't use

gauges), the type 3 cooler has some extra aluminum plates on it that I bent

out to seal it (aka hoover bits), type 3 coolers have 30% more cooling fins

then stock type 1 coolers, all I did to clear the sender boss was cut 2

slits in the rear dog house part and bend the sheet metal up out of the way,

that way if I decide to go another way I can reweld the piece I cut back. I

have installed 3 temp sender units in my engine, I am going to install a

relief screw sender (VDO) and the sender from the oil cooler (outlet side)

from this I should be able to tell where and how the oil is cooled, just

something to do. After I am done with this test I will remove the senders

and gauges. Maybe I'll install a type 4 after this and do the same thing,

(just to see how different the cooling is).
