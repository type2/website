Just put that one back in. It already has a nut on it. 8^)

Seriously, any old 8mm stud should be OK as long as it's the right
length. I usually put red locktite on exhaust studs that come out. I
haven't had that problem since I started using stainless nuts, though. I
know some folks use brass nuts too. I think the main thing is to just
not use plain steel so the stud and nut don't rust together.
