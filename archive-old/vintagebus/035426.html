the 6 digit number on the bottom line of the m code tag is the colour code.

if yours is 424283- the 42 means lotus white  above and below beltline, the 
83 is the interior code for mesh grey.

if it was turquoise, the paint code would be -- wow also 42,  however, that 
was not offered on stanadard micros as turkois/turkois, also turquoise was 
available only march 61 through 64 model year, NLA for a 66.  So by context 
you can tell the 42 means lotus white, not 6turquoise.   BTW mesh grey was 
only offered 65 through 67 model years.
