 The repo duel port center sections are cut for both the narrow base and
wide base carbs, so universal.
  If you get lucky you can sometimes find the narrow base center sections.
These were used on some beetles and I think 71 bus used these also.
  There are only 3 carbs available in the country now unless you get lucky
and find some shop with a 30 pict 1 electric choke model (these were
discontinued over a year ago). Everyone sells the same carbs.
  Bocar and Brosol are the only aftermarket carb manufacturers.
  Some of the new 30/31 carbs come with the adapter.
 There are 2 34 pict 3 available one without the idle down arm (you can
replace the jets in these) and the K Cal model which you cannot replace the
jets in (different size jets)
  The only 30 pict 1 available at this time is the manual choke model
