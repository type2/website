 << I intend to install a 1776cc engine with dual kadrons.  
It  currently has a lightened Scat flywheel attached but 
I have been told that  these are not great for daily drivers.>>

The Kadrons are a nice carb once correctly set up.  As
far as daily or mild performance goes in my book these
are the carbs to use.  Good compromise ($$$) between
dual Webber/Dells and the stock set up.  Fairly easy
to find jets for, as compared to the Webs/Dells, just check
your local Suzi/Yamaha motorcycle dealer.  Am not sure
which one ran Solex carbs instead of Mikunis but the jets
for the Solexs are the same.  

I personally wouldn't recommend using a lightened 
flywheel for daily driving.  Use a full weight balanced
unit and you'll be glad you did.  Makes it easier to stop
and smooths out drivetrain vibrations.  

<<   I want to run 12 volts in the bus so  should I use 
the 6 volt starter with 12 volt solenoid or just swap it up 
to a  straight 12 volt setup? >>

Run a 12 volt flywheel with a 12 volt starter.  Since you 
will be converting over a 6 volt case I'd go ahead and run
an automatic trans starter.  These do not require the 
bushing in the bellhousing, match the flywheel teeth, and
if memory serves, provide more torque to start the engine. 

<< The van in question is an early 64.  >>

I'm going to guess and say that you'll run the 3.875
ring and pinion with a late Bus .82 4th gear set??  If so
you'll be glad you did.  Big time difference at highway
speeds with both noise and engine rpm.  I have and am
building several of this style gearbox for the early split
window busses and my customers are very happy with
the results.
