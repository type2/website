Only if you want the best reference manaul.  

It is a great book, and you would find it useful.    

you need to know about anything hat goes funky with the electric system, and 
it is all there (however, for some stupid reason the bentley reprint has 
black and white diagrams, instead of the colour the original is printed in)

if you have questions about the fuel system,front suspension and steering, 
rear axle and tranny, electrics or body etc. this is an excellent reference.  
particularly in the body chater it outshines any other book.  It is written 
for the dealers, and covers standard, panels, kombis, trucks, ambulaces (ok, 
all models)  and has sections on sunroofs,  steering locks, etc etc .

a number of questions that come up about buses on this list could be answered 
by looking in this book.

and it has really cool stuff and pictures in it.   And you can see nice new 
un crusty buses the way they should be.   And guys in suits working on them.  
good for a laugh.  i enjoyed the air cleaner section.   learned a lot from 
this book, and learn more each time I read it.
