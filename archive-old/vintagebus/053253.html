Hi Joel,
Good point for the main and pilot jetting!  Except one has to go bigger and 
bigger with the air correction jet to get the same result.  Best to figure 
out the pilot jetting first with a main jet a size smaller than what is 
called for.  Then work on the main jet for low end HP with no stumble, and 
this can be complicated by the accellerator nozzle system being off so needs 
to be adjusted or easily adjustable so easily screwed up as in the 34PICT-3 
carb accellerator nozzle.  Then finally the air correction for the higher RPM.

<< Probably the answer is trial and error, I don't think you can count on the 
jet sizes that work in someone else's engine to work the same with yours. I 
would try a 60 pilot first, if that still runs good then try the 55. But 
check the stroke of your fuel pump first if you haven't, I think too high

Again correct.  The jet sizes VW stated always could be off to even the new 
VW engine off the facotry line, but should be close.  So jetting 
experimentation is always best to make sure.
