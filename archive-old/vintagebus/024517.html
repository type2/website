The Pertronix "Ignitor" is an excellent, reliable unit in my opinion. I've
installed them on several cars and impressed with the improved power and
idle. The benefit of never messing with points again is a big plus too. I'm
told that the Mallory units are good too, but are a bit more work to
install. The Pertronix unit is very simple. In any case, be sure the unit
you install is the magnetic type (like Pertronix), not the infrared type
which is more prone to failure.
