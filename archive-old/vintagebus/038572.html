You should build a booth-type setup with a camp box built into the rear
bench (above the engine compartment) containing stuff like a stove, sink,
and maybe a top opening fridge. The booth could have a table that extend
from the wall and sits down flush with the benches to make one big bed.
here's an illustration:
                _____               __
  back of bus<--|    \    _____    / |
             ___|____ \___  |  ___/  |-->front of bus
             |  ___  \   |  |  |     |
             | /   \  \__|_____|_____|


I was going to do this in my Dormobile, but I decided to make a side-facing
bench that runs the length of the bus, behind the driver's seat. This way I
can rebuild snd use my original Dormobile campbox that sits above the
engine. The bench seat will flip out to make a double bed with the back
cushion.
