you do this with RGBoxes NOT attached.
you need an angle finder, check the angle between the spring plate and the 
bus.   then you need to check the book (bentley) to find the correct andle.  
the splines on the torsion bar and the splines on the Spring Plates each have 
different degrees.  Remove the covers and mark where they are now.  are they 
ion the right places?  How are the SPs bad?  some lowering junk?  anyway, you 
do the math and figure out which needds to be turned what way.  it may be one 
notch counter clockwise on the torsion bar, then two to the right on the 
spring plate, or some such.  you have to be very careful to know how many 
notches you are turning what, but the procedure is simple.  and as the tranny 
is already out of the way, should not take long.

I have only done this twice, a few years ago, but it was no big deal.

read the bently several times, till you are familiar with the process.  Once 
thing you MUST do is install BOTH inner bearings before you install the stub 
axle.  if you do not install the upper inner bearing before the stub axle, 
you will find that you have to remove the stub axle to install that bearing, 
which really is a PITA if you already put the nut and plug on.   This is very 
important.
