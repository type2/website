The problem with running the type 3 cool tins on a bus engine is clearances,
if you are useing all the proper tin, heater boxes etc., the type 3 cooling
tin will not clear and you will have to do a lot of trimming to get it to
fit properly. If I am building an engine that is going to use J-pipes then I
will install the cool tin,or you would have to go the NQ route and fabricate
the tin to enclose the J-pipes. I have used the cool tin in stock engine
applications and from my experience it is not worth the hassels and I
couldn't really see the benifits from doing this.
