>  4. I noticed an oil leak below the cylinder, which has
>  orange silicone at the base. What's the procedure for
>  fixing this? 

remove cylinders - marking which is which
remove pistons- marking ditto
- inspect ps and cs
at this point I would inspect the rod bearings

clean cylinder mating area on case and cylinders.  I use sealant, the paper o 
gasket, and more sealant, which I install on the cylinder.
install rods, pistons, cylinders

install heads before the sealant dries.  

so clean, then install rods, working one side at a time, put goo, gasket, goo 
on cylinders, compress rings on piston into cylinder (orient rings and 
pistons properly), installed those ps/cs, install head (with PRTubes- which 
you should have stretched out a tad) do the PRTubes seal look squishy and 
new- or do they have crucnhy bits?

refer to idiot book for details of installation/inspection
