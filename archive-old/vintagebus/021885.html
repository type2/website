>I have never actually met a person that has built an 1835, <
Neither have I.  There are very good reasons that the 1776 is more popular
than the 1835.  Read all the Berg, Hoover, and Fisher stuff to find out why.
I think even Muir advises against it, granted it is one of the "new"
sections in the back which really are not written by him.  The 1776 vs 1835
is the same as the 1586 vs 1641 argument. 

>I would very much like to run an external oil cooler w/ thermostat just for
the added protection if it gets a little spicy. <
Overkill.  This is very easy to add later so I would save the cash for beer
until you knew that there was an overheat problem.  With a properly built
1776 you will not need an oil cooler.  1904 I would though.

>along with a full set of gauges<
Whenever you run an external cooler be sure to add a cylinder head temp
gauge to that list.  You should only need tach(optional), oil press, oil
temp, and cylinder temp, anything else is just a light show.  You can have
cool oil, but the engine load could fry the heads and you would never know
it.  

You may want to consider the type 4 motor swap.  Also I have heard that the
Kubbleschmidt (sp) P&C setup works well, but I have never seen one.  I think
this will give you an indestructible 1700 or 1800....  If my next motor was
not a 1800 type 4 swap I would try this setup.  You may also want to
consider the 1904 which gives you the same thick walled cylinders and more
CCs, granted everything would be custom at this point.  Dave Easterwood is
collecting parts for a 1904 so you may want to talk to him about how to do
it right, Dave is an incredible engine builder.  He also only builds
40HPers, 1586, 1776, and 1904 motors.
