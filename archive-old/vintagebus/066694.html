Dave,

I posted this to the list so that others with experience can chime in.  I
have never used the tar or any sealant before which is probably why I have a
problem with rust now.  The string is to be applied once the seal is on the
window and is wrapped around the channel where the window frame lip sits.
The idea is the be able to press the window into the frame while pulling the
string thus slowly pulling the inner lip over the frame until the window and
seal are fully seated.  I usually wrap the string in the channel two times.
I expect that the cotton string will be covered in tar as I pull it out but
the gloves will keep it off of my skin.  Any tar mess can be cleaned with
mineral spirits, goof-off or lighter fluid.
