The biggest problem I find with VW's is the use of a timing light to set the
timing of a non-stock engine.  On a weekly basis I have VW's coming in the
shop complaining about heat and/or running problems, after I look at it the
first thing I do is advance the timing a few degrees  and that usually
solves the problem. Most non stock VW engines that I see are running to
retarded, which in turn causes them to run hot. The only time I use a timing
light is to check the timing after I have set it by ear, so I know where to
set it if I need to make changes to something and to check the advance if
the dizzy. Using a timing light to set the timing on a NON STOCK engine is a
NO NO, but to each there own. 

ronnie,   I can't prove this to you, but my wifes 65 Type 1 with a NEW MEXI
engine gets 30+ mpg with no heat problems (timing set by ear and test
driving up hills) and as most of you know my DD runs like a bat out of hell,
again no timing light.
