FAG has a catalog online but it doesn't seem to do cross references.  The
URL is hideous, so start at http://www.fag_dot_de/NASApp/Access/index.jsp?lang=1 ,
click "products", then click "catalog".  For some reason, the site is
in English but the catalog is in German -- there are pictures.

You might be able to back into it using the Timken catalog, which does
has a cross reference, lookup by application, and application by part
number (it will show you all the cars that a particular part fits).
http://www.timken_dot_com/industries/automotive/autoaftermarket/timkencatalog/PartCatalog.aspx
