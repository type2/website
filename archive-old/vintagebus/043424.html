>I have a dealer installed lighter on my 66 westy which is not hooked
>up. I see it has an inline fuse, but I do not know what size fuse it
>holds.  Any suggestions?

If the holder is designed for a stock-style (pointy ends) fuse, a
16A (red) fuse would probably be about right.  If it's set up for
a US-style fuse (blunt ends), try a 14A (SFE) or 15A (3AG) fuse.

The end of the fuse holder that doesn't go to the lighter should
go to an unfused, hot-all-the-time wire.  In other words, you
shouldn't be able to make the lighter quit working by pulling
out any of the fuses in the stock fuse box.
