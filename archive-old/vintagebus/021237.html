You're risht, Type 3's had the larger tins that look 
like this (bad ascii art) from the bottom:
 ________________
/  ____     ____  \
| |    |   |    | |
| |____|   |____| |
\_________________/

While T1's and T2's had the smaller, triangular piece. 
  For those that don't know they fit on the underside of 
the barrels in the space between them, against the case, 
pointy bit towards the head.
  Without one of these the cooling air would flow 
straight down and out, causing the barrels to run much 
hotter. When installed they force the air all the way 
around the bottoms of the barrels, cooling much more 
effeceintly. 

More bad ascii, looking down the cylinders:

without:

|  ,--,  ||  ,--,  |
| (    ) || (    ) |
|  '__'  ||  '__'  |
V        VV        V

With:

|  ,--,  ||  ,--,  |
| (    ) || (    ) |
|  '__' /  \ '__'  |
| |----/    \----| |     
V V              V V

  I believe that all type 1 aircooled motors came with 
one style or the other!
