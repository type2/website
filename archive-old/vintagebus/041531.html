I don't know the size right off, but when you buy the tap it should be
specified.  Any competent tool man should be able to help you.  I've found
that many metric taps specify English (or letter or number) drill sizes.

Be very careful when drilling!  The drill will really want to grab and
pull itself in.  Use a variable speed drill that will go very slowly!

Are you sure you really want to do this?  Are you having touble with the
existing studs?  Remember the adage:  If it isn't broken, don't fix it!
