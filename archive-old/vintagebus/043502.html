know if they've done the Type 2 manual yet.  The electronic manuals

scans of *everything* (so you'd have to OCR it first), use sooper-

absolutely crappy navigation.  Oh yeah, Unix and Macs need not
apply.

biological optical-kinetic data transfer (typing it in) and then
using the computer to do what it does well: sorting, cross-
referencing, etc.
