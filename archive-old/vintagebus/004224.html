If you measure the current used by the LED array, divide that into 6.
That will give you the resistor size in ohms that you need to place in
series with the LED's. You will need to calculate the required wattage
also.
formula is w = (I squared) R or E*I
Example:

6V/0.1A = 60 Ohms.
E* I = 6V * .1A = .6W, buy a 1W minimum resistor, recommend using at
least double the calculated size so for this example, buy a 2 watt
unit.
