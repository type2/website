The types of problems that I'd like to know about include (but are not
limited to):

1) Graphics or cosmetic flaws:  i.e. a line break in the text that

2) Part data errors:

4)  Bugs in the way the features work, navigation, etc. Coupon code
(i.e. PROJECT800), want list, etc. (Exactly what were you doing that

5) General comments and suggestions. Be brutally honest. I really want
