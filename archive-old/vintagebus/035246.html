<< Are there any tip's out there for installing the

All the rubber has to be carved, shaped, filed and sanded before 
installation.  Each piece is different - don't assume Left rear and Right 
rear are the same length for example, BTDT.  Only "lubricant" needed is elbow 
grease and lots of it.  I used a screw driver to help push it in.  I have 
done two complete set for some Deluxes I had a few years back.  Takes the 
better part of an hour per trim piece including the shaping.  Oh, the trim 
mounts to the bumper first, then the rubber insert goes in.  Don't forget the 
part.  8-)
