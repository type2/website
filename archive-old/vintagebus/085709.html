Just a note that I now have more issues of the U.K. published "VW Camper &
Bus" magazine in stock.  To my knowledge it is the only English language

you add them to an existing parts order, the incremental shipping cost for

We have only a limited number of each issue, so it will be first come, first
served. As they sell out I will pull them from the website. This magazine is
well worth reading if you haven't had the pleasure yet. Especially if you're
already placing a parts order anyway, I highly recommend adding at least one
of these to your cart.
