Has it got a centre brake light? If it's a '56, it should have tail 
lights like Thom's '55 Westy (check out the picture 6/10/99 on his '55

US version which had a bubble lens) with hexagonal shape centre 
reflector pattern incorporated, round chrome surround with no external 
screws 

'57 model changed to have larger round brake/tail lights - chrome 
surround has external mounting screws visible. From a recent message: 
'The change occurred on 05 April, 1957 at ch # 246-964'
