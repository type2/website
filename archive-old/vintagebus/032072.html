take a dime or penny, or similar sized non US coin, and file it down so it 
will fit the hole, but not fall into the carb.  Pennies are easier to file.  
Make sure the size adjusted coin is clean and grease free, and the carb is 
too.  Mix up some JB weld and attach coin to carb, I like to get artsy and 
leave part of the coin visible as a coin.  Let set before driving.  The one 
in my bug had been in there like this for a few years.
