The difference is night and day. I've done both, and as soon as I'm out of
flux-core wire, I'm converting my Lincoln Weld-Pak 100 to full MIG. I've
already bought the conversion kit, actually, and am literally just burning up
the flux core on crap projects where weld prettiness is not essential. If I
don't use up the flux core before I start welding my next vehicle body, I'm
going to convert anyway.

The major difference with the flux core is that as the core burns, it sputters,
and makes for lousy welds. Stuff that I would weld once with MIG takes me three
times with the flux core to get it right, and my arms get sore from all that
grinding.

Granted, you can control flux core somewhat, but MIG will make a world of
difference.
