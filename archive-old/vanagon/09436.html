 If you are setting the timing at 0 degrees with the hoses attached, you are
 already advanced 5 degrees ahead of stock. If you want to set timing for max
 power, you may want to set it when it is fully advanced: It should be 
 something like 28 - 30 degrees BTDC at 3000 RPM. Advancing much more than 
 that is asking for trouble from detonation when the engine gets hot, IMHO.
