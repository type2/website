   Start with the ignition system test:
   Pull a plug wire off of a spark plug and insert a screw drive into
   the end of the wire. Hold the screw dirve 1/4" away from a good ground
   point. Have someone crank the motor. You should see a spark.
   If you see spark----   
   Check fuel system:
   With a can of starting fluid, spray in to the intake manifold, 1 quick
   shot. Crank engine. It should run for a split second if there is spark.
