Another good possibility might be an INDEX format.  Another active list I am 
on sends the titles once a day and lets you send a batch file back 
requesting only the messages you want.  On that list you simply send a 
command string "set *listname* index".  After that you use the following 
batch file to order:

//       JOB
Database search DD=Orders
//Orders DD *
Select * in <listname>.<message database no.range>
Print <message no. desired>
/*
//       EOJ

I apologize to list members not interested in this meta, but we might 
preserve the participation of some members who just can't deal with *all* 
the mail.
