> 1. What year(s) did the turbo diesel jettaand/or rabbit  sell in the US?
Rabbit '84 (and maybe '83, not sure)
Jetta - '82-92 using same 1.6 from at least 84 on.
Golf '85-92.
> What 	displacement?
1.6litre for all except maybe early 82-83 models. All Golf/Jetta 
models are engine code MF, with hydraulic lifters added in late '86. 
Same code used for hydr, look for sticker on valve cover to indicate 
hydraulic. Also 1V engine code used for "eco-diesel". I don't think 
this was sold in Canada, so I've never seen one. Less power & complex 
exhuast, so I would avoid it.
