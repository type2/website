I've been on the list for less than a year and seen this proposed about 6 
times. The bottom line is that Gerry S owns the list and maintains it the 
way he wants it...and this is the way he wants it.

You might set your subscription to DIGEST mode. Just send a message to 
LISTSERV_at_LENTI.MED.UMN.EDU with the text HELP in the *body* of the 
message. That will return some info, which will include how to set 
vanagon mail to digest mode. Digest mode gives you one message a day with 
all the messages in it.
