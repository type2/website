It is probably ther cooling fan for the condensor.  It gets power all the
time and it thermostatically operated.  The refrigeration cycle in these
things is a complicated process.  Letting the fridge overheat will damage the
internals rendering the unit useless.  Using fridge with the van parked un
level will do same damage.  When using propane, be sure the 12 and 110 volt
switch is off.  The extra heat of both the propane and a heater elemement
will cause eventual damage.
