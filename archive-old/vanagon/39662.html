It's all part of Microsoft's plan for world domination. The sender was 
using Microsoft Exchange to send email. If you're using Exchange, please 
turn off MIME formatting for messages:

Select Tools.Options menu.
Select Services Tab.
Select Internet Mail.
Push Properties button.
Push Message Format button.
Turn off "Use MIME when sending messages"
