What are the readings?  I doubt it is the air flow meter.  If the CO is high,
then the O2 sensor is not working or the engine is not set up properly.  If
the timing is advanced at all beyond spec, it will cause high CO and HC
readings.  If the O2 sensor is working properly, it will overide the airflow
meter.  If the CO is good and the HC is high, then you have an ignition miss,
bad Cat, Vacuum leak, engine problem,  etc.  Follow the Bentley manual set up
procedures exactly and see what happens before buying any parts.  The O2
sensor circuit can be checked with a voltmeter.  It should flucuate between
.2 and .8 volts.    I hope this helps.
