>Speaking of 86 Vanagon model names:
>VW's name for this model was the "Vanagon Camper".  The Westys with the full
>compliment of camping stuff were called the "Vanagon Camper GL".  
>The term "Weekender", while reasonably descriptive, is confusing because VW
>never used this name to describe any Vanagon with a pop-top and they did use
>this name to describe the fold-down rear seat & cushion option on the
>non-camper models.
