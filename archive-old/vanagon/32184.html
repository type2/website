Hello Bill:

The problem seems to be that the EV taken for
modifications by Winnebago is the base Transporter,
sans creature comforts (or in this case, safety
measures).

There seems to be no provision for rear heating
whatsoever - no dual controls, no vent as described
by you.  In my 1988 Westy, not only was there a
core in the box under the rear seat, but there
was a duct running in the housing where the sliding
door ran, with head level outlets.  There is no
equivalent in the EV-Winne.

There is A/C.  Can't wait for the hot, humid weather
to find that it only reaches the front seats!
