             cool air

          /  _________ \__
          | |    |    | ---->

             cool air
           /          \
          /  _________ \__   water in
          | |    |    |  __/
          | |____|____| |  |
          |        ---> |  | ---> hot engine cooling air
          \____________ |__|
                            \ hot? water out
