Have you guys tried the Yellow Pages.  A couple of years ago I took my 
'82 camper to Mann Trailer in Seattle.  They do custom trailers and 
install all kinds of trailer hitches, including custom work.  They 
indicated they could install a hitch, that would handle any load I 
wanted, up to, and beyond, the vehicle maximum load rating!  The maximum 
BUMPER hitch capacity, was another thing however - only a few hundred 
pounds.  The custom hitch would have been welded/bolted directly to the 
vehicle structure in some way.  It was expensive, and it was only going 
to be for occasional use, so I never did get it done and can't vouch for 
how it would have handled.
