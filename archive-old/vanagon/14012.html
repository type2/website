Granted, the transistor would have to dissipate 60 W, but so would any power
resistor that could be used for this application, but the fact that the
transistor controls the voltage at 6V to the wiper motor (or more if you wanted
the motor to run a little faster), means that the motor won't bog down under
any condition (like when it is pushing aside wet snow).  There are many
transistors that can easily handle these requirements as long as they are
sufficiently heat sunk (as in mounted to the chassis), the main problem is that
the case has to be isolated, so a mica or sil-therm insulator must be used
along with shoulder washers when mounting the unit.

Another idea I read in this string regarded two motors in series.  I tried this
some twenty years ago when I converted my bug to twelve volts when I was in
high school.  This doesn't work, as the motor that is driving the wipers has a
larger load than the dummy motor, so the voltage drop gradually shifts to the
unloaded motor, making it run faster, which increases the voltage drop, which
makes it run faster, which, well you get the picture, the dummy motor ends up
running reeeaaallll fast and the wipers stop moving.

Since that first conversion, I have converted a '65 and a '64 type II to twelve
volts and in both cases merely had to scour the junk yard for a VW wiper motor
that was nearly the same as the old 6V unit. I found one in a Ghia and the
other in a Bus.  It was really surprising how many different styles of motors
that VW used, but there were many similarities, so that I could find something
that nearly bolted right in.  I have always been happy with this solution and
am a firm believer that the best way to convert the wipers is by changing the
motor, as reliability is very important here.  An added advantage is that many
of the twelve volt motors are two speed, so not only do they work right, but
they work better than the old single speed six volt wipers.
