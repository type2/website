> is a Kombi? the name has been around since the split-window buses. surely it

 I thought it was the German for "Station wagon" (or "Estate" in the UK), this
is the translation given in the Aircooled FAQ from r.a.vw, anyway:

Kombi = Station Wagon
Kaefer = Insect
Kubelwagen = "bucket car"

 etc.

 My understanding is that it is a type designator rather than a trim level.

 The Caravelles were higher-spec. buses in the UK, I think some were used for
aftermarket conversions as well so there are, I think, Camper-Caravelles as
well as buses.
