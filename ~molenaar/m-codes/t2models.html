Type 2 models (1968-1979)

The Options plate has a 6 digit number which indicates the type 2 model:

XXXX XX This number is always a "2" because the van is a "type 2" (the beetle is called a type 1)
XXXX XX The sub model, a closed van, a kombi, a pick up...
XXXX XX More specific: LHD/RHD, cargo door left/right,...
XXXX XX Indicates a certain m-code, you must look it up in the list to see which one.
XXXX XX Type of engine (a beetle (type 1) or pancake (type 4) engine)
XXXX XX Type of transmission (manual or automatic)

The list of models is very long. To speed up download time I had to cut it into pieces. Select the model you have below (XXXX XX), to get to the more specific list (XXXX XX).

Special thanks to Ron van Ness who spend time and money to send me copies of his VW microfiche. All the model info comes from this fiche.

If you find any errors, if you have more info or if you can't find your model in this list then don't hesitate to send me a message: molenari@hotmail.com
