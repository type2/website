T2 Export destinations (1968-1979)

A destination code of letters is for an export destination outside Germany. Destinations inside Germany are printed as a number. For example "EN" stands for "England", "004" for "West-Berlin" (probably). What the German codes exactly say is not clear. It could stand for a specific VW-dealer or for a certain town or distribution point.

Until approximately Chassis number 2x9 151 000 of modelyear 1969 there were no codes for the German market. In that case the space of this code remains empty.

If you find a code which is not yet in this list, let me know. Also if you know the meaning of certain unidentified codes: molenari@hotmail.com.
