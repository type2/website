Model codes

The model code identifies the vehicle type, if it is a LHD or an RHD version, and in some cases it contains certain M-codes. The list is sorted by vehicle type. The vehicle type can be read by the first two digits in the model code.

Vehicle types:
21  - Panelvan
22  - Microbus (from 1974 onwards also Microbus Deluxe)
23  - Kombi
24  - Microbus Deluxe
26  - Pick-up
27  - Ambulance
