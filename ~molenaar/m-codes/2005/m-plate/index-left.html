How to read the M-plate?

DD D  (E)
MMM MMM MMM MMM MMM
YZ MMM MMM MMM MMM
XX TTTT PPPPUU CCCCCCC
layout model years 1968-1970

CC CCC CCC  (E)
MMM MMM MMM MMM MMM
PPPPUU  MMM MMM MMM MMM MMM
DD D RRRR  XX  TTTT YZ

layout model years 1970-1979

DD D - 
 scheduled prod. date
E - 
 built in Emden instead of Hannover
YZ - 
 aggregate; engine(Y) and gearbox(Z)
MMM - 
 M-code or group code
XX - 
 destination
TTTT - 
 model
PPPPUU - 
 paint(P) and interior(U)
CCCCCCC - 
 chassis number (VIN)
RRRR - 
 prod. planning code
