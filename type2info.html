The type2 mailing list covers aircooled buses made from 1968 through 1983. If you have a split-window bus go to the vintagebus list instead.

If you have a watercooled bus go to vanagon.com instead.

The type2 list normally generates 30-60 messages per day. A digest is available. The charter contains the list rules. Please consult the Library and the Archive before trying to post questions to the list.

If you need help, the best place to write is the on-duty administrator. The unindicted co-administrators of the type2 list are:
