Subject: Vanagon Web stuff update


I rolled over the Vanagon list archive today.  There were over 600 messages
posted since I started the new archive on 29 November.  What a prolific
bunch of writers.  :-) If you access the archive now you will only see a few
messages.  If you click on the "Other mail archives" link however, you will
get the old archive.  Click on it again from there and you hit the terminal
link in the chain.  Let me know if you have any trouble.

I have subscription info for about eight or ten VW-ish lists (including the
Audi, Porsche, and kit-cars lists).  I also have some new image pointers,
including one to John Gladu's Bozobus.

If anybody out there has a mailing list archive they are maintaining with
Hypermail, I have a Python script that you can use to roll it over.  Python
is a very cool interpreted object-oriented language.  Check it out if you're
into things like perl or tcl but are looking for something suitable for
building more complex applications.
