I'm sure you'll get a lot of responses on this...

There were two generations of engines (well simplified a lot) put into 
type 2s.  There really isn't a type 2 engine (before the wasserboxer anyway).

At first the type 2 shared an engine with the type 1.  Later (after '72 I 
believe) it shared an engine with the type 4 (914,411,412).
