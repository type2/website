>In the VW fuel injected engine, there is a cold-start valve which

        This is simply incorrect.  Or only 1/2 correct...

        It's the cold *start* valve, not the cold *running* valve.

        The thermo-time switch, as you mentioned, does control the
grounding of this "fifth" injector.  However, the + side is wired to the
starter switch.  This cold *start* valve only gets current if the following
are BOTH true:

        a)  the starter is being engaged, AND
        b)  the thermo-time switch is cold

The two failure modes of the cold *start* system are:

        1)  never opens --> hard to start the car, then runs fine
        2)  leaks all the time --> car always runs rich

        The "choke" functions are controlled by the brain & temp sensors,
which enrichen the mixture (by changing injector duration) whenever the car
is *running* cold.  The cold start system, is well, the cold *start*
system.
