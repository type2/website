The consensus seems to be BFG Radial All Terrain T/A, size 27 850 14. The
best price I've seen is mail order for $78 ea incl tax + $48 shipping for a
set (call 1 800 589 6789). Be sure to get a 5th tire for spare, this is a
syncro requirement. This is primariily an off-road tire, but has good
highway characteristics according to many on the list with this tire. I'm
running Michelin LTX 195 75 14 on my syncro Westy, but I don't do a lot of
steep muddy roads.
