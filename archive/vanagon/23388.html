I've been carrying a hand winch for awhile.  It has about 16' of 5/16" cable
on a drum with a ratcheting handle.  Very common and I've seen really cheap
ones for $12, better ones run about $35-50.  Of course 5/16" cable isn't
good for too much, maybe 1500 lbs using the block.  If you're trying to pull
your 5000 lb. bus back onto the road from 6' down an embankment or
something, there would need to be some serious hardware available, like
1/2"-5/8" cable and a 2-ton chain motor.  Or, hey, how about a 4000 lb.
winch?  Maybe drive more carefully on that Ramshorn summit road?
