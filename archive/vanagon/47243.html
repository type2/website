Yeah, If the original poster is still reading this thread, here goes.

The VIN should look something like this:

WV2YB0259GG123456

W=Germany
V=Volkswagen
1=Multi purpose vehicle
Y=Deluxe van, windows
B=Waterboxer engine
0=Something about passenger restraints
25=Van body style
9=secret check digit number
G=Model Year, F=85, G=86, H=87, J=88, K=89, L=90, M=91(No USA 1988 Syncros)
G=Graz Austria factory, H=Hanaover Germany factory (Syncros=G)
123456=Sequential production number
