OK, here goes.  The first line is the date of manufacture:  Feb 27,
1961.  The second line is the option code line.  I don't know what
315 is but 180 is "no middle seat" (this is from the latest NEATO
newsletter, a list of almost all option codes).  The third line is
the export destination code (anyone have a list of these?), then 
the model (yes you have a 221), then a paint code, then the
VIN.
