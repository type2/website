Gail,

With only 4 non-Westies in the survey, there is not enough data to draw conclusions about 
differences between the two configurations.

The 185R14 reinforced Michelin is probably a better selection for passenger buses.  For 
Westies that might see some "off the road" use (ie, camping) the Yokohama LT195/75R14 light 
truck tire is probably better.  Also, the Yokohama is a slightly bigger tire, so can better 
handle the conversions greater weight.
