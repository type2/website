The rear facing seats and fold up table started (in non-Westy models) with
the '87 Wolfsburg Edition, & continued with the '88 Wolfsburg Edition.  These
were so popular (in relative Vanagon terms) that VW offered the Carat in '89,
'90, & '91 that offered the same interior package in a distinct model that
was available all year long.
