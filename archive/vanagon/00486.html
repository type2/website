I've seen a standard rain-gutter setup for this purpose.  False rain-gutter
flanges were mounted high onto the sides of the pop-top, say 3" tall by
4 or 5" long:

	===================================
        |                                 |
	----------------------------------
        |                                 |
        |                                 |
        |                                 |
        |    ---                   ---    |
        |    | | <-- Bolt Holes -> | |    |
        |    ---                   ---    |
        |                                 |
        |                                 |
	-----------------------------------

A standard rain-gutter mount roof rack (the usual two cross bars)
would mount to these.  Off season, the flanges themselves could be
removed.
