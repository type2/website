In California, the two right lanes are where the trucks by law must 
drive. Unless you are about to exit the freeway, stay out of those
lanes.

Since most freeways have four to six lanes on each side, here is
the algorithm for a VW bus:

Lanes are numbered from the middle of the freeway, so the fast lane is
always lane 1.

switch (number_of_lanes)
{
  case 7:
    lane_to_drive_in = 5;
    break;

  case 6:
    lane_to_drive_in = 4;
    break;

  case 5:
    lane_to_drive_in = 3;
    break;

  case 4:
    lane_to_drive_in = 2;
    break;

  case 3:
    lane_to_drive_in = 2; /* go ahead and fight with the trucks */
    break;

  case 2:
    lane_to_drive_in = 1; /* it's not a real freeway */
    break;

  case 1:
    lane_to_drive_in = 1; /* it's not a freeway at all */
    break;

}
