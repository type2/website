     (for example). HOWEVER...running uudecode on something like
      _vw.gif_ that is really /etc/passwd ... but if you permissions
      are set correctly...this will not happen.
      
  c. Getting code off BBS's is always risky...I always v_scan every thing
     that comes within running distance of my machine...
