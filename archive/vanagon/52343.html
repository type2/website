In order to correctly set the valve timing and injector pump timing you'll
need the tools listed.  Diesel injection pumps are very sensitive to timing
adjustments, so when you change the head gasket and realign the drive belt
you'll have to use a dial indicator to set the timing correctly.  It cost me
$90 bucks for the VW tools alone, I adapted a dial indicator that I had and
fabricated a mount to set the timing.  Sorry, but not the normal tools that
you would probably have in your toolbox.
