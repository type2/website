>Tim--69campmobile

Birch doorskin (1/8"). I dunno what the original topcoat was, but I've been
using satin polyurethane. Birch ply is so cheap, I don't see any reason to
try to re-finish the old stuff. By the time you've sanded through the
cracks, you're through the first lamination. If I had it to do over, and
still had a shop, I'd probably shoot laquer on both sides of the ply BEFORE
I installed it. Make sure you finish both sides to avoid
telegraphing/cracking.
