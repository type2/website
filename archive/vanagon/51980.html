If the ignition is on right timing, then the ignition mark (2) printed
on the front half of the pulley (seen into driving direction)
must be in line with the crease/split of the engine case if it is
lighted by a stroboscop lamp.


If you want to adjust the top return point (Obere Todpunkt), then the
top return mark (1) has to be turned to the crease/split of the engine
case.
