symptom is engine will idle high, like 1500-2000 rpm, if you go rap on

one of the hoses, spray in GUMOUT carb cleaner, spray in WD40 liberally.

since.  Did it on the G60 about 2000 miles ago, still OK.  Note these
are covered under emission wartranty 5/50 but 7/70? in CA? so go
to dealer if you can prove it is acting up if still under the mileage
