The vintagebus mailing list covers aircooled buses made from 1949 through 1967. If you have a bay-window bus go to the type2 list instead.

If you have a watercooled bus go to vanagon.com instead.

The vintagebus list normally generates 20-50 messages per day. A digest is available. The charter contains the list rules. Please consult the Library and the Archive before trying to post questions to the list.
