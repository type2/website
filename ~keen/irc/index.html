Jatoro Java IRC Client for Type2.Com


Select "Setup" and put a nickname in the "Nick" box. Press "Connect" and wait.
Note that you can only use the servers www.type2.com or irc.type2.com, it won't work with others.

Commands

NB: ccc=channel nnn=nick

Commands to the individual nick are also available from command window. To show command window, double click on a nick in the nick list.


/join #ccc	Join the channel. You can omit #ccc when invited by someone.
/w		See who are in the current channel
/who #ccc	See who are in the specified channel
/i nnn		See the person's detail
/ii nnn		See the person's detail including idle time
/topic ttt       Set the channel topic to ttt (when you are op)
/mode #ccc xxx  Channel mode command (op, deop, etc)
/msg nnn mmmm   Send a private message mmmm to nnn
/quit :mmmm	Quit with a message mmmm
/me mmmm	Shows your action mmmm to the current channel
/version nnn	Find out what irc client nnn is using. (omitting nnn applies the whole channel)
/time nnn	Find out nnn's local time (omitting nnn applies the whole channel)
/ping nnn	Find out the distance to nnn (omitting nnn applies the whole channel)
/finger nnn	Finger nnn (omitting nnn applies the whole channel)
/clientinfo nnn	Available CTCP list of nnn's irc client (omitting nnn applies the whole channel)
/userinfo nnn	Info of nnn (omitting nnn applies the whole channel)
/showurl uuu    Show the URL with new browser popped up
/swctcp	        Switch CTCP flood protection (default true)	
/swshowurl      Switch showurl-enable (default true)
/swtstamp       Switch timestamp (default true)
/paste          Send the contents of the TextArea in the setup window
etc..
